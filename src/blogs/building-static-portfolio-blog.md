# Building a Static Portfolio Blog

A static blog keeps the portfolio simple, portable, and easy to deploy.

The content lives in markdown files, while supporting metadata such as tags, summaries, categories, and reading time.

Manual JSON files can be used for RSS links, recent reads, and pinned or featured metadata without adding a backend.
